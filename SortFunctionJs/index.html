Sort function JavaScript

Custom sort function that works exactly like the javascript sort function. (More or less) :D

the function 'sortIt(list, [conditonal function]); can be called for sorting array of numbers or strings with just 1 arguement, ie the list, while it can be called with another argument function for custom objects.

Eg. for custom object array var objects = [{x:1, y:'xy'},{x:5, y:'cd'},{x:2, y:'mq'}]; , you may call the function to sort it on the basis of x.

It goes like: sortIt(objects, function(a, b){ return a.x-b.x;})

Here a and b are two elements of the array. if the function returns -1, the array will keep b at lower index and a at higher index, and vice versa.
